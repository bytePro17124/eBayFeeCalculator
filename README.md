# eBayFeeCalculator

How to use:  
* right click on *eBayFeeCalculator-win32.zip*  
* Choose _Save link as..._  
* Unzip to desired location and run the .exe  
* Make shortcuts to the .exe as desired (right click and _pin to start_ or whatever you like)
